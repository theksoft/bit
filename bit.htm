Tools

Draw areas: hex, rectangle, square, rhombus, equilateral, isoceles & right-angle triangles, ellipse, circle, polygon.

Draw grid area - hex, rectangle & circular - griding an existing selected area.

Change strict grid scope: grided areas are strictly limited to inner grid scope or not.

Change grid alignment: grided areas can be aligned differently depending on alternate alignment availability.

Grid spacing : Modify internal grid spacing

href, alt, title, id : View or set selected area parameters. Display them on area hovering.

/ Show internal grid ordering & allow selecting different ordering methods

/ Resize & align selected areas on first selected area - centered horizontally & vertically or aligned on left, top, right & bottom side.
